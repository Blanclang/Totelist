# Totelist
**vue+vuex构建的Todolist**

----------

- 项目启动
	
    	// 将项目下载到本地，git bash到项目文件夹

		// 安装依赖包
		npm install

		// 启动项目
		npm run dev
- 基本功能
	
 	- 添加事件
 	- 切换事件状态
 	- 事件历史记录
 	- 事件记录查询
 	- 删除记录
